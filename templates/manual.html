1. サービス概要

{{ app_name }} は、社会科（主に世界史）の学習を支援するWebアプリケーションです。 一問一答形式のクイズ、論述問題のAI添削、年代順並び替え、地図クイズ、RPGモードなど 多彩な学習機能を搭載しています。

クラス・部屋単位でユーザーを管理し、担当者（マネージャー）が出題範囲や設定を柔軟にコントロールできます。

2. アカウントとログイン

ログイン方法

ログインページから、部屋番号・出席番号・個人パスワードを入力してログインします。

「ログインを記憶する」にチェックを入れると、次回からパスワード入力を省略できます。

パスワード・ユーザー名の変更

  • パスワード変更: ログインページの「パスワード変更」リンクから変更ができます。
  • ユーザー名変更: ログイン後、画面右上の設定（歯車アイコン）からパスワードを変更できます。
  • パスワードリセット: パスワードを忘れた場合は、ログインページの「パスワードを忘れた場合」リンクからリセットをリクエストできます。
初回ログイン時にパスワードの変更を求められる場合があります。

3. 一問一答（メインクイズ）

ログイン後のメイン画面から利用できる、{{ app_name }} の中核となる学習機能です。

出題設定

  • 出題数の選択: 10問 / 20問 / 50問 / 全問 から選択できます。
  • 章・単元の選択: 出題範囲を章ごとに選択します。各章を展開して単元単位での選択も可能です。「全選択」ボタンで章内の全単元を一括選択できます。
  • フィルター:「未回答のみ」などのフィルターで出題範囲を絞り込めます。
利用可能な章・単元は、担当者（マネージャー）が設定した範囲に限定されます。

出題モード

モード 説明
通常モード 選択した範囲からランダムに出題されます。
克服モード 過去に間違えた苦手問題を集中的に出題します。

解答方法

  1. 問題文が表示されます。
  2. 「答えを見る」ボタンをタップすると、正解が表示されます。
  3. 自分の回答と照らし合わせて、「正解」または「不正解」ボタンを押して自己判定します。
  4. 全問終了後、結果画面に正答率がグラフで表示されます。
文字サイズは画面右下の「＋」「−」ボタンで調整できます（パソコン・タブレットのみ）。
結果画面にある「論述チャレンジ」から関連する論述問題に直接挑戦できます。

音声入力

マイクボタンを使って、声で解答することができます。

  1. 問題カード画面に表示される赤いマイクボタンをタップします。
  2. 解答を声に出して読み上げます（日本語認識）。
  3. AIが音声を認識し、正解と自動的に照合して正誤判定を行います。
音声入力は Chrome や Safari など、Web Speech API に対応したブラウザでのみ利用可能です。

4. 今日の10問

毎日更新される日替わりクイズです。全ユーザー共通の問題が出題されます。

  • メイン画面に「今日の10問」カードが表示されます。
  • 1日1回挑戦可能です。
  • 結果は記録され、部屋全体での成績を確認できます。
今日の10問は管理者が設定・配信する問題が出題されます。

5. 苦手問題

過去の学習結果をもとに、正答率の低い問題を「苦手問題」として自動集計します。

  • メニューの「苦手問題」からアクセスできます。
  • 苦手問題の一覧をカテゴリ別に確認できます。
  • 一問一答の「克服モード」と連携して、苦手問題に集中して取り組めます。

6. 論述問題演習

大学入試レベルの論述問題に挑戦し、AIによる自動添削を受けられる機能です。

問題の閲覧

  • 章別一覧: 学習中の章に対応した論述問題を表示します。
  • 大学別一覧: カスタム検索ページで、出典大学ごとに問題を絞り込めます。キーワード検索やフィルター機能も搭載しています。
  • 各問題には出典大学名・出題年・難易度が表示されます。
閲覧可能な問題は、担当者（マネージャー）が各部屋ごとに公開設定を行っています。

AI添削機能

Google Gemini APIを活用したAI添削機能です。

  1. 論述問題の画面下部にある解答欄にテキストを入力します。
  2. 「AI添削を受ける」ボタンをクリックすると、AIが採点・添削します。
  3. 改善点などがフィードバックされます。
AI添削の結果はあくまで参考情報です。実際の入試採点とは異なる場合があります。

手書きOCR

手書きの解答をカメラで撮影し、テキストに変換する機能です。

  1. 論述問題画面の「プリント読込」タブをタップします。
  2. 手書き解答の画像をアップロードします。
  3. AIが文字認識（OCR）を行い、テキストに変換します。
  4. 変換結果を確認・修正した上で、AI添削に進めます。

添削リクエスト

AI添削とは別に、管理者に直接添削をリクエストすることも可能です。

  • 「添削依頼」から解答を送信します。
  • 管理者が手動で添削を行い、コメントが返されます。
  • 「添削履歴」ページから過去の添削結果を確認できます。

7. 年代順並び替え問題

歴史上の出来事を正しい年代順に並び替える問題です。

  1. セクション（章）を選択します。
  2. 表示された歴史イベントカードをドラッグ＆ドロップで年代順に並べます。
  3. 「解答する」ボタンで判定されます。
難易度 説明
易 基本的な出来事の順序問題
標準 標準的な難易度
難 細かい年代の知識が必要
最難関 上級者向けの高難度問題

8. 地図クイズ

歴史地図上の場所を特定するクイズです。

  1. ジャンルを選択してクイズを開始します。
  2. 地図と問題文が表示されます。
  3. 地図上の正しい場所をクリック（タップ）して解答します。
  4. 正解・不正解がフィードバックされ、正しい位置が表示されます。

全問正解すると「習得」として記録されます。

目次に戻る

9. RPGモード

クイズに正解して敵を倒していく、ゲーミフィケーション機能です。

  1. 特定のスコアに達成した際に表示されるRPGバナーをタップするとバトルが開始されます。
  2. 敵キャラクターが表示され、クイズに正解するとダメージを与えられます。
  3. 敵を倒すと「称号」を獲得でき、プロフィールに装着できます。
  4. 倒した敵は「再戦」で何度でも挑戦できます。
獲得した称号は、設定によりナビゲーションバーのユーザー名の横に表示されます。

10. 進捗管理・成績確認

学習の進捗をグラフやランキングで可視化できます。

  • 個人進捗: 章ごとの正答率、学習済み問題数をグラフで表示します。
  • ランキング: 同じ部屋のユーザーとの成績比較ができます（部屋内ランキング）。
  • 月次スコア: 月ごとの学習成績の推移を確認できます。
  • 成績詳細: 章別・単元別の詳細な正答率を確認できます。

進捗ページへは、ナビゲーションバーのグラフアイコンからアクセスできます。

11. 今日のニュース

時事ニュースを社会科学習に活かす機能です。

  • 外部報道機関のRSSフィードから最新ニュースを取得・表示します。
  • 各ニュースにはAIが生成した「学びのポイント」が付与されます。
  • アーカイブ: 過去のニュースをカレンダー形式で振り返ることができます。
  • 検索: キーワードで過去のニュースを検索できます。
ニュース記事のリンクは外部サイトに遷移します。「学びのポイント」は当サービスのAIが独自に作成した学習支援コンテンツです。

15. 通知機能

ブラウザのプッシュ通知に対応しています。

  • 設定メニューから通知のON/OFFを切り替えられます。
  • 新しいお知らせ、添削結果の返却などをプッシュ通知で受け取れます。
  • 通知を有効にするには、ブラウザの通知許可が必要です。
PWAとしてホーム画面に追加すると、より確実に通知を受け取れます。

18. よくある質問

Q. パスワードを忘れました

A. ログインページの「パスワードを忘れた方」リンクからリセットをリクエストしてください。管理者の承認後、新しいパスワードを設定できます。

Q. 出題範囲が限定されています

A. 部屋の担当者（マネージャー）が出題範囲を設定しています。授業の進度に合わせて範囲が拡張されますので、担当者にご確認ください。

Q. AI添削が利用できません

A. AI添削機能は管理者がONに設定している場合のみ利用可能です。また、サーバー側の設定（APIキー）が必要です。利用できない場合は管理者にお問い合わせください。

Q. RPGモードが表示されません

A. RPGモードは管理者が機能をONにしている部屋でのみ利用できます。担当者または管理者にお問い合わせください。

Q. 通知が届きません

A. 以下を確認してください：

  1. ブラウザの通知設定で当サイトの通知が「許可」になっているか
  2. 設定メニューで通知がONになっているか
  3. 端末の通知設定でブラウザの通知がONになっているか

Q. スマートフォンで文字が小さいです

A. ピンチイン・ピンチアウトで拡大縮小できます。また、PWAとしてホーム画面に追加すると、より快適に利用できます。
